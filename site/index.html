Guarden PW Manager

Guarden PW Manager is a secure password and private data manager for storing website logins, bank account details, subscription records, and personal notes in one encrypted local vault.

The app is designed around local control: your sensitive data stays on your device, protected with AES-256 encryption and biometric unlock. Google OAuth is used only when you choose the optional encrypted Google Drive backup feature.

Read Privacy Policy Contact Support

What the app does

  • Stores passwords and sensitive records in an encrypted local vault.
  • Unlocks with your master password and supported biometric authentication.
  • Lets you back up your encrypted vault to your own Google Drive if you opt in.
  • Does not require a Guarden cloud account to use the core password manager.

Purpose

Guarden PW Manager helps users organize and protect private credentials without depending on a remote server for day-to-day vault access.

Google OAuth usage

OAuth access is limited to the optional encrypted backup flow so the app can create, update, and restore the backup file stored in the user's own Google Drive.

Privacy

The privacy policy explains what data stays on-device, how Google Drive backup works, and how to contact support for questions about data handling.

Stored data types

  • Website usernames and passwords
  • Bank account and card-related records
  • Subscription billing details
  • Private notes added by the user

Security model

  • Offline-first local storage
  • AES-256 encryption before storage or backup
  • Biometric unlock on supported devices
  • No required remote Guarden account
