Privacy Policy for Guarden - Premium Privacy

Last updated: March 9, 2026

Guarden - Premium Privacy is an offline-first password and private data manager. This Privacy Policy explains what information the app handles, how optional Google Drive backup works, and how users can contact us about privacy questions.

1. Core app data

Guarden - Premium Privacy stores password entries, bank account records, subscription records, and private notes locally on the user's device. The app is designed so that core vault usage does not require a remote Guarden account.

2. Encryption and local protection

Sensitive vault content is protected with encryption before storage. On supported devices, users can also enable biometric authentication to help protect access to the vault.

3. Google OAuth and Google Drive backup

Guarden - Premium Privacy requests Google OAuth access only if the user chooses the optional encrypted backup feature. That access is used to create, update, download, or restore the encrypted vault backup file in the user's own Google Drive.

The app's Google Drive access is limited to the files it creates for the user's backup flow. The backup file remains encrypted before upload.

4. Data sharing

We do not sell vault data. Guarden - Premium Privacy is built so that private vault contents remain under the user's control on the device or in the user's own encrypted backup file.

5. User choices

Users may choose not to use Google Drive backup and continue using the offline-first password manager locally on the device.

6. Contact

For privacy questions or support requests, contact jackdrawer90@gmail.com.
