Privacy Policy for Guarden - Premium Privacy

Guarden - Premium Privacy is an offline-first password and private data manager. This page explains what the app handles, how optional Google Drive backup works, and how users can contact us about privacy questions.

Last updated: March 9, 2026

Back to Home Terms of Service Contact Support

Policy Summary

  • Core vault records stay on the user's device.
  • Encryption protects content before storage or backup.
  • Google OAuth is used only for optional encrypted Google Drive backup.
  • Users can continue using the app without enabling cloud backup.

1. Core app data

Guarden - Premium Privacy stores password entries, bank account records, subscription records, and private notes locally on the user's device. Core vault usage does not require a remote Guarden account.

2. Encryption and local protection

Sensitive vault content is protected with encryption before storage. On supported devices, users can also enable biometric authentication to help protect access to the vault.

3. Google OAuth and Google Drive backup

Guarden - Premium Privacy requests Google OAuth access only if the user chooses the optional encrypted backup feature. That access is used to create, update, download, or restore the encrypted vault backup file in the user's own Google Drive.

4. Data sharing

We do not sell vault data. Private vault contents remain under the user's control on the device or in the user's own encrypted backup file.

5. User choices

Users may choose not to use Google Drive backup and continue using the offline-first password manager locally on the device.
