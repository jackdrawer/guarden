- Wrap closeDatabase and deleteDatabase in try-catch, throw DatabaseError on failures
- Migrate StorageError thrown on line 45 to use imported error type

**secure_storage_service.dart:**
- Import lib/errors/app_errors.dart (replace existing inline StorageError definition)
- Remove old StorageError class definition (move to centralized app_errors.dart)
- Wrap all flutter_secure_storage read/write operations in try-catch
- Catch PlatformException and throw StorageError("Secure storage access failed", "Couldn't access secure storage. Please re-enter master password.", canRetry: true, action: "re-enter password")
- Ensure all existing StorageError throws use new imported type

**biometric_service.dart:**
- Import lib/errors/app_errors.dart
- Wrap local_auth.authenticate() in try-catch
- Catch PlatformException and throw BiometricError("Biometric auth failed", "Biometric authentication failed. Try again or use master password.", canRetry: true, action: "retry")
- Log technical errors to console (development only)

**Why these services first:** These are critical path services used in auth flow and data persistence. Proper error handling here prevents app crashes on login/save operations.

- Import lib/errors/app_errors.dart
- Create helper method `Future<T> _retryWithBackoff<T>(Future<T> Function() operation, int maxAttempts)`

- Wrap HTTP calls in _retryWithBackoff
- On final retry failure, throw NetworkError("Pwned API request failed", "Couldn't check password security. Check your internet.", canRetry: true, action: "retry")
- Catch SocketException, TimeoutException, HttpException → convert to NetworkError
- Log retry attempts to console (development): "Retrying pwned check... (attempt X of 3)"

**purchase_service.dart:**
- Import lib/errors/app_errors.dart
- Wrap RevenueCat API calls (getOfferings, purchasePackage) in try-catch

**logo_service.dart:**
- Import lib/errors/app_errors.dart
- Wrap cached_network_image operations in try-catch
- On network failures, return default icon/placeholder (graceful degradation)
