**1. Create GoRouter config** (`lib/router.dart`)

```dart
import 'package:flutter_riverpod/flutter_riverpod.dart';
import 'package:go_router/go_router.dart';
```

Define route paths as constants:
```dart
class AppRoutes {
  static const setup = '/setup';
  static const seedPhrase = '/seed-phrase';
  static const login = '/login';
  static const recovery = '/recovery';
  static const home = '/';
}
```

Create `routerProvider` as `Provider<GoRouter>` that takes `ref`:

```dart
final routerProvider = Provider<GoRouter>((ref) {
  final authState = ref.watch(authStateProvider);

  return GoRouter(
    initialLocation: AppRoutes.login,
    redirect: (context, state) {
      final currentPath = state.matchedLocation;

      // Auth state based redirects
      if (authState == AuthState.setupRequired) {
        if (currentPath != AppRoutes.setup) return AppRoutes.setup;
      } else if (authState == AuthState.unauthenticated || authState == AuthState.locked) {
        if (currentPath != AppRoutes.login && currentPath != AppRoutes.recovery) {
          return AppRoutes.login;
        }
      } else if (authState == AuthState.authenticated) {
        if (currentPath == AppRoutes.login || currentPath == AppRoutes.setup) {
          return AppRoutes.home;
        }
      }
      // AuthState.initial -> no redirect, show loading or check auth

      return null; // no redirect
    },
    routes: [
      GoRoute(
        path: AppRoutes.home,
        builder: (context, state) => const HomeScreen(),
      ),
      GoRoute(
        path: AppRoutes.setup,
        builder: (context, state) => const MasterPasswordSetupScreen(),
      ),
      GoRoute(
        path: AppRoutes.seedPhrase,
        builder: (context, state) {
          final seedPhrase = state.extra as String;
          return SeedPhraseScreen(seedPhrase: seedPhrase);
        },
      ),
      GoRoute(
        path: AppRoutes.login,
        builder: (context, state) => const LoginScreen(),
      ),
      GoRoute(
        path: AppRoutes.recovery,
        builder: (context, state) => const RecoveryScreen(),
      ),
    ],
  );
});
```

```dart
import 'package:flutter/material.dart';
import 'package:flutter_riverpod/flutter_riverpod.dart';
import 'package:guarden/router.dart';
import 'package:guarden/services/auth_service.dart';

void main() {
  WidgetsFlutterBinding.ensureInitialized();
  runApp(const ProviderScope(child: GuardenApp()));
}

class GuardenApp extends ConsumerStatefulWidget {
  const GuardenApp({super.key});

  @override
  ConsumerState<GuardenApp> createState() => _GuardenAppState();
}

class _GuardenAppState extends ConsumerState<GuardenApp> {
  @override
  void initState() {
    super.initState();
    // Check auth state on app launch
    Future.microtask(() {
      ref.read(authServiceProvider).checkAuthState();
    });
  }

  @override
  Widget build(BuildContext context) {
    final router = ref.watch(routerProvider);

    return MaterialApp.router(
      title: 'Guarden',
      debugShowCheckedModeBanner: false,
      theme: ThemeData(
        colorScheme: ColorScheme.fromSeed(
          seedColor: const Color(0xFF2E7D32), // Dark green - security/trust color
          brightness: Brightness.light,
        ),
        useMaterial3: true,
        inputDecorationTheme: const InputDecorationTheme(
          border: OutlineInputBorder(),
          contentPadding: EdgeInsets.symmetric(horizontal: 16, vertical: 12),
        ),
        elevatedButtonTheme: ElevatedButtonThemeData(
          style: ElevatedButton.styleFrom(
            minimumSize: const Size(double.infinity, 48),
            shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(8)),
          ),
        ),
      ),
      routerConfig: router,
    );
  }
}
```

The app flow:
1. App starts -> GuardenApp initState calls checkAuthState()
2. checkAuthState reads secure storage -> sets AuthState
3. GoRouter redirect watches authStateProvider -> routes accordingly
4. setupRequired -> /setup -> user creates password -> /seed-phrase -> confirms -> /
5. unauthenticated/locked -> /login -> user enters password (or biometric) -> /
6. authenticated -> / (home)

Integration with AppLifecycleService: In _GuardenAppState, also initialize the lifecycle service. Add a listener on isLockedProvider that sets authState to locked when true:

```dart
@override
void initState() {
  super.initState();
  Future.microtask(() {
    ref.read(authServiceProvider).checkAuthState();
    // Initialize lifecycle observer
    ref.read(appLifecycleProvider);
    // Listen to lock state
    ref.listen(isLockedProvider, (prev, next) {
      if (next == true) {
        ref.read(authServiceProvider).lock();
        ref.read(isLockedProvider.notifier).state = false; // reset
      }
    });
  });
}
```

IMPORTANT: When user successfully authenticates (login or biometric), the authState changes to authenticated, which triggers GoRouter redirect to home. No manual navigation needed from login screen beyond setting state.
