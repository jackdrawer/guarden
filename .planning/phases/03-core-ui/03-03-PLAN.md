**Step 2: Create Neumorphic widget library**
All widgets use AppColors from `lib/theme/app_colors.dart`.

- `lib/widgets/neumorphic/neumorphic_container.dart`: A Container wrapper that applies `AppColors.neumorphicShadows` (light top-left, dark bottom-right). Accept `child`, `padding`, `borderRadius` (default 16), `width`, `height` params. Use `AppColors.background` as base color. Wrap in a `Container` with `BoxDecoration`.

- `lib/widgets/neumorphic/neumorphic_button.dart`: A tappable neumorphic container. On press, switch from outer shadows to a subtler shadow (reduced offset/blur) to simulate pressed state. Accept `onPressed`, `child`, `padding`. Use `GestureDetector` + `AnimatedContainer` for smooth transition. Optional `color` param for accent-colored buttons (e.g., save buttons use `AppColors.primaryAccent`).

- `lib/widgets/neumorphic/neumorphic_textfield.dart`: A TextField wrapped in a neumorphic inset-style container (use inner shadows: dark top-left inset, light bottom-right inset for "sunken" look). Apply `AppColors.background` fill, `AppColors.textPrimary` text style, `AppColors.textSecondary` hint style. Accept `controller`, `hintText`, `obscureText`, `keyboardType`, `onChanged`, `prefixIcon`, `suffixIcon`, `validator`.

**Step 3: Create HomeShell with bottom navigation**
Create `lib/screens/home/home_shell.dart`:
- A `StatelessWidget` that receives `navigationShell` from GoRouter's `StatefulShellRoute.indexedStack`.
- Bottom navigation with 4 tabs using BottomNavigationBar:
  - Index 0: Dashboard (icon: Icons.dashboard_rounded, label: "Ana Sayfa")
  - Index 1: Banks (icon: Icons.account_balance_rounded, label: "Bankalar")
  - Index 2: Subscriptions (icon: Icons.subscriptions_rounded, label: "Abonelikler")
  - Index 3: Passwords (icon: Icons.lock_rounded, label: "Sifreler")
- Style: `backgroundColor: AppColors.background`, `selectedItemColor: AppColors.primaryAccent`, `unselectedItemColor: AppColors.textSecondary`, `type: BottomNavigationBarType.fixed`, `elevation: 0`.
- Wrap the BottomNavigationBar area in a Container with `AppColors.neumorphicShadows` applied on top edge for neumorphic separation.
- `onTap: (index) => navigationShell.goBranch(index)`, `currentIndex: navigationShell.currentIndex`.

**Step 4: Update lib/router.dart**
- Remove the placeholder `DashboardScreen` class entirely.
- Import all new screen files from their respective paths.
- Replace the single `/` GoRoute with a `StatefulShellRoute.indexedStack`:
  - `builder`: returns `HomeShell(navigationShell: navigationShell)`
  - Branch 0: path `/`, routes: `GoRoute(path: '/', builder: => DashboardScreen())`
  - Branch 1: path `/banks`, routes: `GoRoute(path: '/banks', builder: => BankAccountsScreen())`, sub-routes: `GoRoute(path: 'add', builder: => BankAccountFormScreen())`, `GoRoute(path: 'edit/:id', builder: => BankAccountFormScreen(accountId: state.pathParameters['id']))`
  - Branch 2: path `/subscriptions`, routes: `GoRoute(path: '/subscriptions', builder: => SubscriptionsScreen())`, sub-routes: `GoRoute(path: 'add', builder: => SubscriptionFormScreen())`, `GoRoute(path: 'edit/:id', builder: => SubscriptionFormScreen(subscriptionId: state.pathParameters['id']))`
  - Branch 3: path `/passwords`, routes: `GoRoute(path: '/passwords', builder: => WebPasswordsScreen())`, sub-routes: `GoRoute(path: 'add', builder: => WebPasswordFormScreen())`, `GoRoute(path: 'edit/:id', builder: => WebPasswordFormScreen(passwordId: state.pathParameters['id']))`
- Keep ALL existing auth redirect logic intact.
- Keep existing routes for `/login`, `/welcome`, `/onboarding`, `/biometric-optin` OUTSIDE the shell route.

NOTE: The CRUD screen files imported here will be created in plans 03-04 and 03-05. For this plan to compile, create minimal placeholder files for each screen (just a StatelessWidget with a Scaffold returning Center text). Plans 03-04 and 03-05 will overwrite these with full implementations.
