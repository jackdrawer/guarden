**Bank Accounts List (`lib/screens/bank/bank_accounts_screen.dart`):**

- FAB: `NeumorphicButton` with "+" icon, navigates to `/banks/add`.
- Body: `ref.watch(bankAccountProvider)` -> if empty, show centered "Henuz banka hesabi eklenmedi" with an add button. Otherwise `ListView.separated`.
- Each item: `NeumorphicContainer` with `InkWell` (tap -> `/banks/edit/${account.id}`). Row layout: LogoService `getLogoWidget(account.url ?? account.bankName)` on the left (48x48), Column with bankName (bold, textPrimary) and "Her ${account.periodMonths} ayda degistir" subtitle (textSecondary), trailing: days-until-change chip or icon.
- Swipe to delete: Wrap in `Dismissible(key: ValueKey(account.id), direction: DismissDirection.endToStart, background: red container with delete icon, confirmDismiss: show AlertDialog "Bu hesabi silmek istediginize emin misiniz?", onDismissed: ref.read(bankAccountProvider.notifier).deleteBankAccount(account.id))`.

**Bank Account Form (`lib/screens/bank/bank_account_form_screen.dart`):**
Replace the placeholder with a `ConsumerStatefulWidget`. Accept optional `String? accountId` constructor param. If non-null, this is edit mode.
- Use `TextEditingController`s for each field. In `initState`: if editing, find the account from provider list by ID and pre-fill controllers.
- Form fields (wrapped in `Form` widget with `GlobalKey<FormState>`):
  - `NeumorphicTextField(hintText: "Banka Adi", controller: nameCtrl, validator: required)`
  - `NeumorphicTextField(hintText: "URL / Domain", controller: urlCtrl)`
  - `NeumorphicTextField(hintText: "Sifre", controller: passwordCtrl, obscureText: true, suffixIcon: visibility toggle)`
  - **Period dropdown:** `NeumorphicContainer` wrapping `DropdownButtonFormField<int>(value: selectedPeriod, items: [3,6,9,12].map((m) => DropdownMenuItem(value: m, child: Text("$m Ay"))).toList(), onChanged: ..., decoration: InputDecoration(labelText: "Sifre Degisim Periyodu"))`.
  - **Last Changed date:** `NeumorphicButton` showing formatted date. `onPressed: () async { final picked = await showDatePicker(...); if (picked != null) setState(() => lastChanged = picked); }`.
  - `NeumorphicTextField(hintText: "Notlar", controller: notesCtrl, keyboardType: TextInputType.multiline, maxLines: 3)`
- Save button: Full-width `NeumorphicButton` with `AppColors.primaryAccent` background. On pressed: validate form, create/update `BankAccount` object, call provider's add or update method, `context.pop()`.
- AppBar title: editing ? "Hesabi Duzenle" : "Yeni Banka Hesabi".
- Dispose all TextEditingControllers in dispose().
